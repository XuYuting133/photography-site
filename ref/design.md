# 街拍与风光摄影作品集网站 · UI设计文档 (design.md)

> **版本**：v1.0  
> **更新日期**：2026-07-12  
> **适用场景**：个人摄影师作品集网站，以街拍 (Street Photography) 与风光 (Landscape Photography) 作品为核心

---

## 1. 总体设计原则

### 1.1 核心设计哲学：作品至上 (Photo-First)

摄影作品集网站的核心使命只有一个：**让作品本身成为绝对的视觉焦点**。所有设计元素——色彩、字体、布局、交互——都应为烘托和展示摄影作品服务，而非喧宾夺主。

### 1.2 极简主义 (Minimalism)

极简是摄影网站永恒的经典风格。其核心在于：
- **大量留白**：在照片周围留出充足的空白空间，将观众的注意力自然引向作品。
- **减少干扰**：精简导航、去除冗余装饰，降低访客的认知负荷。
- **内容为王**：让每一张照片都有呼吸的空间，而非被信息淹没。

### 1.3 叙事性 (Storytelling)

无论是街拍的“决定性瞬间”还是风光的“自然史诗”，好的摄影网站不应只是一本作品目录，而应像一本精心编排的摄影集 (Monograph)。设计应引导访客以一种特定的节奏和情绪浏览作品，创造沉浸式的观看体验。

### 1.4 沉浸式体验 (Immersive Experience)

利用全屏、大画幅等设计手法，让访客仿佛置身于作品之中。一个前沿的设计理念是让 **“图像成为界面”** (Image is the Interface)，即用全幅的摄影作品本身作为导航和交互的背景。

## 2. 色彩方案 (Color Palette)

### 2.1 核心策略：克制与衬托

色彩的作用是衬托而非 competing。建议采用 **“中性背景 + 作品自然色彩”** 的策略。

| 风格类型 | 主色调 | 辅色调 | 适用场景 |
| :--- | :--- | :--- | :--- |
| **纯净极简 (推荐)** | 纯白 (#FFFFFF) / 浅灰 (#F5F5F5) | 黑色 (#1A1A1A) / 深灰 | **风光摄影**：凸显自然色彩的丰富与通透，营造宁静、通透的观看氛围。 |
| **深邃暗调 (推荐)** | 纯黑 (#000000) / 深灰 (#222222) | 白色 (#FFFFFF) / 暖金 | **街拍摄影**：营造沉浸感、戏剧张力和都市的深邃氛围，让光影对比更突出。 |
| **大地色系** | 米白、暖灰 | 赭石、森林绿 | **自然/风光摄影**：与自然主题呼应，营造温暖、质朴的品牌感。 |

### 2.2 配色建议
- **保持简洁**：全站主色不超过2-3种。
- **突出CTA**：唯一的强调色（如品牌色）可保留给“联系”或“购买”等关键按钮。

## 3. 排版 (Typography)

### 3.1 核心策略：清晰与对比

字体是设计中最有力的叙事工具之一。目标是 **“读起来毫不费力，看起来充满格调”**。

| 风格 | 字体选择建议 | 说明 |
| :--- | :--- | :--- |
| **标题 (Headings)** | **粗体无衬线体** (如 Inter, Helvetica Neue) 或 **粗衬线体** (如 Playfair Display) | 粗体大字能与摄影作品形成强烈的视觉对比，本身也成为设计元素。 |
| **正文 (Body)** | **细体无衬线体** (如 Roboto, Open Sans) | 保证在大量留白和图片背景上的极高可读性。 |
| **字号对比** | **强烈对比** | 例如，标题用120pt超大字重，正文用12-16pt轻盈字重，模仿高端杂志的编辑风格。 |

### 3.2 风格建议
- **街拍摄影**：可尝试带有粗粝感、工业感或复古感的字体，与都市主题呼应。
- **风光摄影**：偏好优雅、干净、带有自然气息的字体。

---

## 4. 布局与交互 (Layout & Interaction)

### 4.1 核心策略：呼吸感与引导

布局设计应引导访客的视线，控制观看的节奏。

| 设计元素 | 建议 | 说明 |
| :--- | :--- | :--- |
| **网格布局 (Grid)** | 使用**非对称网格**或**马赛克布局** | 打破传统齐整网格的呆板，为作品集注入更多活力与艺术感。 |
| **全屏展示 (Full-Screen / Full-Bleed)** | 首页或系列作品使用全屏大图 | 创造无边界、沉浸式的视觉体验。 |
| **分屏布局 (Half and Half)** | 一侧为竖幅作品，另一侧为文字 | 营造杂志般的编辑感，适合呈现作品背后的故事或项目陈述。 |
| **边框与细线 (Borders & Thin Lines)** | 为照片添加细边框 | 营造画廊般的展示效果，让每张作品都像一幅独立的艺术品。 |
| **导航 (Navigation)** | **极简、隐蔽** | 采用汉堡菜单、纯文字链接或极简图标，避免干扰作品展示。 |

### 4.2 交互建议
- **滚动动画**：平滑的滚动和微妙的视差效果可以增加浏览的趣味性。
- **悬停效果**：在作品缩略图上提供微妙的放大或信息提示反馈。

## 5. 街拍摄影网站专项设计建议

街拍摄影的魅力在于捕捉城市中真实、偶然、富有戏剧性的瞬间。

- **视觉基调**：推荐 **“暗调、高对比”** 风格。深色背景能凸显街拍作品中常见的光影对比和戏剧张力。
- **内容编排**：**强调序列与节奏**。街拍作品不应随机排列，而应有意识地编排，在紧张、复杂的画面与安静、简约的画面之间交替，形成视觉上的“张弛有度”。
- **文字处理**：**摒弃繁琐的图片说明**。街拍作品的力量在于其本身的神秘感和多义性。用一段简短诗意的“项目陈述”开篇，然后让照片自己说话。
- **关于我**：讲述你与街头的故事。是什么吸引你走向特定的街区？你想通过镜头表达什么？

## 6. 风光摄影网站专项设计建议

风光摄影的魅力在于展现自然的壮丽、静谧与宏大。

- **视觉基调**：推荐 **“明亮、通透、自然”** 的风格。浅色或大地色背景能让自然的丰富色彩成为绝对主角。
- **布局重点**：**全屏展示是首选**。让壮丽的山川湖海充满整个屏幕，给访客以身临其境之感。
- **色彩搭配**：可从你的作品中提取主色调，应用到网站的背景或强调色上，形成视觉上的呼应。
- **叙事方式**：可以按“拍摄项目”或“旅行目的地”来组织作品，讲述每一次深入自然的探索故事。

## 7. 技术实现备忘 (结合Vercel + 无数据库方案)

- **性能优先**：摄影网站对加载速度要求极高。务必使用 **Vercel** 的全球CDN和 **WebP** 等现代图片格式。
- **图片管理**：强烈建议使用 **Vercel Blob** 或 **Cloudinary** 等专业图片托管服务，实现图片与代码分离，并享受自动优化和CDN加速。
- **响应式设计**：确保在移动设备上，全屏大图和复杂布局依然优雅、流畅。
- **暗色模式**：提供**深色模式**已成为现代网站的标准配置，尤其适合摄影作品展示。
